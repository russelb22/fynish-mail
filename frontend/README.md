# Fynish Frontend

React + Vite + TypeScript frontend for the Fynish local review queue UI.

## Local Development

```bash
npm install
npm run dev
```

The frontend runs at [http://127.0.0.1:5173](http://127.0.0.1:5173) and proxies API calls to the local backend.

## Build

```bash
npm run build
```

## Auth Helper Tests

```bash
node --test auth.test.mjs
```
